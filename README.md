# Thyges bogstavapp

En legende tegneapp til danske bogstaver, lavet til Thyge.

## Version 5

- Materialer, klistermærker, fortryd og slet er samlet ved titlen.
- De to ekstra vandrette værktøjsmenuer er fjernet for at give større tegneområde.
- Langsommere modellervoks-overgang mellem bogstaver.
- Både ét-etagers og to-etagers lille `a` vises sammen med stort `A`.
- Ny malerpensel med obligatorisk valg af farve, form og størrelse.
- Nyt træværktøj med obligatorisk valg af farve, bladform og antal stammer.
- Ét konfigureret træ vokser ud fra hver træstreg.
- Alle klistermærker vælges fra én samlet oversigt.
- Der kan tegnes oven på placerede klistermærker.
- Det store bogstav holdes helt synligt, inklusive bunden.

## Version 6

- Træværktøjet har valg mellem eg, birk, bøg og ahorn.
- Trætyperne har forskellige barkmønstre og bladformer.
- Stammetykkelse kan vælges som smal, mellem eller tyk.
- Stammetykkelsen er tydeligst ved roden og aftager opad.
- Pynt indeholder kirsebær, kastanje og bog.
- Kastanje og bog findes både som nød og i pigget skal.
- Versioner bruger fremover kun hele tal.
